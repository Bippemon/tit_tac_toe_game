# tit_tac_toe_game

Denna repo är testa samlas in alla olika version av spel som ni eleverna har skapat och vi använda pull request for kod granskning. 

* Skapa en Fork av denna repo
* Klona er repo 
* Skapa en mapp med ditt namn (utan mellanslag och svenska bokstäver)
* Lägga in filen med tit_tac_toe_game_1.cpp för första del uppgift
* Lägga in filen med tit_tac_toe_game_2.cpp för andra del uppgift
* Lägga in filen med tit_tac_toe_game_3.cpp för tredje del uppgift

* Pusha upp din repo 
* Skapa en pull requests
* Jag kommenterar er pull requests och ni förbättra koden
* Sedan merga jag in er pull requests
